# Snap to Learn 智慧日語情境學習平台

GitHub repository name: `115502`

第 115502 組系統已整理為可操作的靜態前端工作台，包含互動任務、狀態管理與本機瀏覽器資料保存。

## 系統能力

- 將日常照片轉為日語單字卡、假名、例句與可複習的個人教材。
- 提供 AI 日語會話、文法修正、假名輔助與快捷回覆，降低開口壓力。
- 以成就徽章、學習小組、J-Pts 與週任務維持長期學習節奏。
- 支援 Premium 權益、學習歷程與前後端分離服務架構。

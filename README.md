# Snap to Learn 智慧日語情境學習平台

GitHub repository name: `115502`

此公開展示版依據第 115502 組學生文件的系統目的、功能與預期成果重新設計，重點是讓版型、視覺語言與操作情境更貼近該系統的產品定位。

## 展示重點

- 創造生活即教材的學習價值，將日常照片轉為客製化日語教材。
- 打造零壓力 AI 日語對話環境，提供文法修正、假名輔助與快捷回覆。
- 以成就徽章、學習小組與成長數據提升 App 開啟率與長期黏著度。
- 驗證 J-Pts、Premium 訂閱與前後端分離架構的商業可行性。
